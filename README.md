# FPGA-Based-Radar-for-Signal-Detection-in-ADAS-using-nuScences-
Developing RTL-based FPGA radar signal processing modules in Verilog to enable real-time object detection for automotive ADAS  applications using the nuScenes dataset. 
  
## My Contributions

* Implemented FPGA radar processing modules using Verilog HDL
* Worked on BRAM storage, range filtering, grid mapping, and clustering
* Implemented centroid detection and distance estimation logic
* Integrated and tested the hardware pipeline in Vivado
* Performed simulation and validation on PYNQ-Z2 FPGA
* Worked with radar point cloud data from the nuScenes dataset

## Tools & Technologies

* Verilog HDL
* Xilinx Vivado
* PYNQ-Z2 FPGA
* Python
* NumPy
* Matplotlib
* Jupyter Notebook
* nuScenes Dataset
